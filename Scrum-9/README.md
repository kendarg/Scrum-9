# ⚽ Pasión Mundialista

**Pasión Mundialista** es un blog web desarrollado con **HTML5** y **CSS3** cuyo objetivo es presentar información sobre los cuartos de final de la Copa Mundial de la FIFA 2026 mediante una interfaz moderna, intuitiva y responsive.

## 📌 Características

- Diseño moderno.
- Menú de navegación con efecto hover.
- Uso de HTML semántico.
- Maquetación con Flexbox.
- Paleta de colores personalizada.

## 🛠️ Tecnologías utilizadas

- HTML5
- CSS3
